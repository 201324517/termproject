* 게임 소개

Tetris - 테트리스는 여러가지 모양이 다른 블록이 떨어지면서 쌓여가는데 이를 한줄이 다채워지면
그 블록이 없어지면서 점수가 올라가는 게임이다. 블록을 정렬하지 않고 막 쌓게 되면 블록이 다 쌓여서
게임이 끝나게 된다.


* 게임 Tutorial

1. 게임 시작 1번버튼을 누른다.
2. i-모양바꾸기 j-왼쪽 l-오른쪽 k-아래 키를 이용하여 테트리스 게임을 즐긴다.
3. 기록을 하기위해 id를 입력한다.
4. 이 때까지의 기록을 확인할 수도 있고 내 id를 검색하여 기록을 확인할 수도있다.
5. 4를 입력한후 엔터를 하면 게임이 종료되고 다시 리눅스 상으로 넘어간다.
